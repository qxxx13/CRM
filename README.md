## Запуск проекта

Чтобы запустить проект:

- Клонируйте репозиторий на свой компьютер
- В каталоге проекта запустите `npm i`
- Запустите `npm run dev`
- Откройте [localhost:3000](http://localhost:3000/), чтобы посмотреть его в браузере
